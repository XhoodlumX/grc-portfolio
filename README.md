# GRC Portfolio — Demonstration Project

**Fictional scenario:** BrightPath Recruitment Ltd, a 25-person recruitment
agency in Reading, UK. It holds CVs, passport scans and salary details of
~8,000 candidates in a cloud CRM, runs on Microsoft 365 with staff using
personal phones for email, and has recently adopted an AI tool that screens
CVs automatically. It has no security team.

This portfolio contains three governance, risk and compliance artefacts
built for that scenario:

| # | Artefact | Frameworks used |
|---|----------|-----------------|
| 1 | Information security risk assessment & risk register | ISO 31000:2018, 5×5 matrix |
| 2 | Policy pack (InfoSec, Access Control, Acceptable Use) | ISO/IEC 27001:2022 Annex A |
| 3 | DPIA for the AI CV-screening tool | ICO DPIA template, UK GDPR, EU AI Act |

> **Disclaimer:** BrightPath Recruitment Ltd is fictional. This is a
> demonstration project created to show GRC working methods. No real
> company, personal data or client engagement is represented.
